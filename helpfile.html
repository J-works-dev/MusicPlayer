Music Player Help File

How to Use

  • ADD Button: Add a Song inth the Playlist
  • Search Button: You can search songs from the list with name of song
  • Delete Button: You can delete a song selected from the list
  • Save CSV: It is saving whole playlist to a CSV file
  • Load CSV: It is loading playlist from saved CSV file
  • Play Button: It paly first song or selected song
  • Previous Button: It Play previous song
  • Next Button: It play next song
  • Slider: You can seek song position.
  • Double Click list: It automatically play list
